## Visão Geral

Tendo escrito um dos posts mais compartilhados no medium.com ("_Como sacar o 401K do seu pai sem ele saber_"), a FamilyFund nos procurou querendo construir seu próprio blog.

O blog ajudaria a direcionar novo tráfego para o site deles e serviria como um centro de recursos para usuários já tentando explorar sua rede por dinheiro. Como era tão importante que eles fossem donos de seu próprio conteúdo, decidimos que uma solução on-premise seria a melhor.

Instalamos 24 Mac Minis comprados no craigslist no armário de armazenamento do escritório deles. Uma máquina seria usada para o servidor web e outra para o servidor de build. As outras 22 eram para redundância e para realizar DDOS em squarespace.com a cada poucos meses para mantê-los alertas.

Para otimizar o tráfego de busca, usamos uma técnica inovadora. Cada post tem um post sombra visível apenas para web crawlers que é alguma variação de _"Gary Vee está procurando investir em novos fundadores"_. Como abelhas ao mel.
