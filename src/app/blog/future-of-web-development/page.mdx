## 1. Desenvolvimento Assistido por IA

Com o lançamento do Github Copilot em 2022, a indústria teve seu primeiro vislumbre de como seria ter o Stack Overflow diretamente integrado ao seu IDE. O Copilot proporcionou a milhares de desenvolvedores o que eles sempre desejaram: uma ferramenta poderosa para aumentar a produtividade e a qualidade do código.

![](./pilot.jpg)

Em 2023, podemos esperar que esses assistentes se tornem mais sofisticados e que isso tenha efeitos em toda a indústria.

Prevemos que o tráfego para documentações online diminuirá à medida que os desenvolvedores perceberem que podem obter informações rapidamente através desses assistentes. Também esperamos que surjam comunidades dedicadas ao compartilhamento de prompts eficazes para IA, tornando-se recursos valiosos para desenvolvedores.

## 2. Padrões de Renderização

Renderizar no servidor ou não? Em 2022, empresas líderes em hospedagem decidiram que, em vez de fazer essa escolha uma vez para todo o aplicativo, agora você precisará decidir cada vez que escrever um novo componente.

Como o desenvolvimento front-end está se tornando mais complexo, os mesmos profissionais que escrevem CSS agora precisarão entender como funcionam o Streaming SSR e a Hidratação Progressiva.

![](./server.jpg)

Em 2023, podemos esperar que os frameworks adotem padrões de renderização cada vez mais granulares, culminando em renderização por linha (PLR) ainda este ano. Também podemos esperar que as vagas para Engenheiros de Confiabilidade de Renderização atinjam um número recorde.

## 3. Runtimes JS

Porque escolher um runtime JS era uma das poucas áreas onde um desenvolvedor não estava paralisado com escolhas, no início de 2020, o criador do Node nos deu algo novo para considerar. O lançamento do Deno e do Bun anunciou a mutação final do JavaScript em uma linguagem que pode realmente rodar em qualquer lugar.

Esses novos runtimes JS significam que agora podemos servir HTML mais rápido do que nunca. Por exemplo, reduzimos o Time to First Byte (TTFB) deste blog para níveis impressionantes, garantindo uma experiência de usuário excepcional.

![](./laptop.jpg)

Em 2023, podemos esperar o lançamento de runtimes JS ainda mais rápidos e especializados, incluindo soluções promissoras projetadas para reduzir os tempos de inicialização em diversos dispositivos. Todos esses avanços prometem tornar o futuro do desenvolvimento web verdadeiramente empolgante.
